Конспект: CSS-властивості блоків
До головного меню

display — багатоцільова властивість яка задає глобальні параметри тегу.

Не успадковується.

  1. block — елемент відображається як блочний (за замовченням для блочних об'єктів);
  2. inline — елемент відображається як рядковий (за замовченням для рядкових об'єктів);
  3. inline-block — елемент відображається як рядковий, але з можливістю використання усіх CSS-властивостей (блочно-рядковий);
  4. none — елемент не виводиться в браузер. Інші елементи займають його місце. Не бере участь в обробці браузером.
До головного меню

box-sizing — застосовується для зміни алгоритму обчислення величин.

  1. content-box — (за замовченням) додає відступи до розміру об'єкта;
  2. border-box — значення розмірів враховують відступи.
До головного меню
Відступи

padding — внутрішній відступ. Використовується для відступів всередину блоку (елементу), а не між блоками (елементами). Значення можуть бути лише додатними, тобто більше або рівне нулю.

Графічне відображення відступу padding

Відступи всередині об'єкту.

Від кордонів об'єкта до його контенту.

Не успадковується.

При вказанні значення у відсотках, відсоток береться від ширини батьківського об'єкта, навіть для вертикальних відступів.

margin — зовнішній відступ. Використовується для відступів між блоками (елементами), рекомендується робити відступи зверху вниз (margin-bottom). Значення можуть бути від'ємними!.

Графічне відображення відступу margin

Відступи зовні об'єкту.

Відступ margin повинен бути на одному рівні вкладеності елементів, тобто від елементів першого рівня вкладеності до елементів першого рівня. Заборонено робити відступи з другого рівня вкладеності до першого рівня як це вказано нижче.
